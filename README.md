# Sammen mot barnekreft.

Dette repoet er for foreningen 🌈 Sammen mot barnekreft. Målet er å samle inn penger til barnekreftforeningen ved å arrangere kunstauksjon, med mer. 

## Gjøre endringer
Gå til `/public` mappen. Alle filene du finner her representerer en url på siden.

### Legge til en ny side
For å opprette en ny side kan du legge til en ny fil ved å trykke *add file* oppe til høyre. Kall denne for det du ønsker. Bruk `-` om den består av flere ord og bruk filformatet `.md` for markdown. F.eks. `om-oss.md`

Enn så lenge så må jeg legge det til i menyen/ det gjøres i koden, men om det blir et behov for mange flere sider så kan vi finne en løsning 🫰

## Kjøre koden lokalt? Start her

Du må ha node på maskinen din, anbefaler å bruke [nvm (node version manager)](https://github.com/nvm-sh/nvm)

Intaller

```bash
npm i
```

Kjør opp dev serveren med

```bash
npm run dev
```

Åpne [http://localhost:3000](http://localhost:3000)

### Deploy

Bare å oppdatere koden, så deployes siden som en statisk side på github pages
